This page has been styled by a responsive style sheet, which was inspired by Twitter Bootstrap.

Responsive Layouts

This responsive style sheet can display up to three (3) columns that automatically arrange themselves based on the type and orientation of a device.

There is a two (2) column layout with an offset: with the appropriate viewport size, this will display the first column wider than the second column.
